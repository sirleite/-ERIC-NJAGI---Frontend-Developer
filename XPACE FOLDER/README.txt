# SpaceX Data

## Installation

1. Clone the repository:

```bash
git clone <repository_url>
```

2. Install the dependencies:

```bash
npm install
```
3. Start the development server:

```bash
npm start
```
4. Open the application in the browser:

```bash
http://localhost:3000
```


## Usage

Search for SpaceX data using the search form.
